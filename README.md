# QtOpenSSL
QtOpenSSL is a wrapping of [OpenSSL](https://github.com/OpenSSL/OpenSSL) for [Qt](https://qt.io).  

# How does it work?
The API functions of this library only call some OpenSSL functions and cast them into Qt's built-in type.
